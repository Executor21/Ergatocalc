# 📋 Εγχειρίδιο Χρήστη - Ergatocalc

## 🎯 Εισαγωγή

Καλώς ήρθατε στο **Ergatocalc** - τον έξυπνο υπολογιστή μισθοδοσίας που κάνει τους πολύπλοκους υπολογισμούς να μοιάζουν παιχνίδι! 

Αν είστε εργατοτεχνίτης, λογιστής, ή απλά κάποιος που θέλει να ξέρει ακριβώς πόσο "κοστίζει" ο μισθός του, τότε βρήκατε το σωστό εργαλείο. Το Ergatocalc μετατρέπει τις περίπλοκες ασφαλιστικές εισφορές, τους φόρους και τις προσαυξήσεις σε απλά, εύκολα να κατανοήσετε αποτελέσματα.

**Γιατί αξίζει;** Γιατί αντί να χάνετε ώρες με αριθμομηχανές και excel, τώρα μπορείτε να έχετε ολοκληρωμένους υπολογισμούς μισθοδοσίας σε δευτερόλεπτα. Και το καλύτερο; Δεν χρειάζεται να είστε λογιστής για να το χρησιμοποιήσετε!

## 💻 Εγκατάσταση

### 🔧 Προϋποθέσεις Συστήματος
- **Λειτουργικό Σύστημα**: Windows 10 ή νεότερο
- **Αποθήκευση**: Τουλάχιστον 50 MB ελεύθερος χώρος
- **Δικαιώματα**: Συνιστάται εκτέλεση ως Διαχειριστής (Administrator)

### 📦 Οδηγίες Εγκατάστασης
1. **Εκτέλεση Εγκαταστάτη**: Κάντε διπλό κλικ στο αρχείο `Ergatocalc Setup.exe`
2. **Ακολουθήστε τον Οδηγό**: 
   - Πατήστε "Next" για να ξεκινήσει η εγκατάσταση
   - Επιλέξτε φάκελο εγκατάστασης (προτείνεται ο προεπιλεγμένος)
   - Επιλέξτε αν θέλετε συντόμευση στην επιφάνεια εργασίας
3. **Ολοκλήρωση**: Πατήστε "Finish" και... είστε έτοιμοι!

> 💡 **Συμβουλή**: Η εγκατάσταση στον προεπιλεγμένο φάκελο (C:\Ergatocalc) αποφεύγει πιθανά προβλήματα δικαιωμάτων.

### ⚡ Πρώτη Εκκίνηση
- Εκκινήστε το πρόγραμμα από το Start Menu ή την επιφάνεια εργασίας
- Αν σας ζητηθούν δικαιώματα διαχειριστή, πατήστε "Ναι" - δεν πρόκειται να χακάρει το ψυγείο σας (υπόσχεση! 😊)
- Στην πρώτη εκκίνηση δημιουργούνται αυτόματα τα αρχεία ρυθμίσεων

## 🚀 Ξεκινώντας

### 🎨 Η Κύρια Οθόνη
Το Ergatocalc έχει 4 βασικές καρτέλες:

1. **🧮 Μισθοδοσία** - Μηνιαίοι υπολογισμοί μισθών
2. **🎄 Δώρο Χριστουγέννων** - Ειδικοί υπολογισμοί για το χριστουγεννιάτικο δώρο
3. **🐣 Δώρο Πάσχα** - Υπολογισμοί για το πασχαλινό δώρο  
4. **🏖️ Επίδομα Αδείας** - Υπολογισμοί επιδόματος αδείας

### ⚙️ Βασικές Ρυθμίσεις
- **Αρχείο Ρυθμίσεων**: Το πρόγραμμα δημιουργεί αυτόματα το `Wage_Config.cfg` στον φάκελο εγκατάστασης
- **Προεπιλογές**: Όλες οι βασικές τιμές (ημερήσιοι μισθοί, συντελεστές) έχουν ήδη ρυθμιστεί
- **Προσαρμογή**: Μπορείτε να τροποποιήσετε τις τιμές απευθείας στο αρχείο ρυθμίσεων

## 💰 Χρήση του Προγράμματος

### 🧮 Καρτέλα Μισθοδοσίας
**Βασικά Στοιχεία:**
- Επιλέγετε ημέρες εργασίας (26 προεπιλογή)
- Σημειώνετε αν είστε παντρεμένος (+10% στον βασικό μισθό)
- Ορίζετε τριετίες (0-6, +5% ανά τριετία)
- Επιλέγετε μικτό ημερομίσθιο και βασικό ημερομίσθιο
- Επιλέγετε κωδικό ασφάλισης (101-108)

**Προσαυξήσεις:**
- Υπερεργασία (+20%)
- Νόμιμη υπερωρία (+40%) 
- Νυχτερινή εργασία (+25%)
- Κυριακές/Αργίες (+75%)
- Και πολλές άλλες συνδυαστικές προσαυξήσεις

### 🎄 Καρτέλα Δώρου Χριστουγέννων
- Επιλέγετε ημέρες απασχόλησης (1/5 - 31/12)
- Το σύστημα υπολογίζει αυτόματα τον συντελεστή δώρου
- Προσθέτετε προσαυξήσεις από Μάιο-Νοέμβριο
- Βλέπετε αναλυτικά καθαρό δώρο

### 🐣 Καρτέλα Δώρου Πάσχα
- Παρόμοια λειτουργία με το χριστουγεννιάτικο δώρο
- Περίοδος απασχόλησης: 1/1 - 30/4
- Διαφορετικοί συντελεστές υπολογισμού

### 🏖️ Καρτέλα Επιδόματος Αδείας
- Απλοί υπολογισμοί για επίδομα αδείας
- Επιλέγετε ημέρες απασχόλησης (1-13)
- Άμεση προβολή καθαρού ποσού

### 💾 Αποθήκευση Αποτελεσμάτων
- Πατήστε "ΑΠΟΘΗΚΕΥΣΗ" σε οποιαδήποτε καρτέλα
- Τα αρχεία αποθηκεύονται στον φάκελο "Μισθοδοσία"
- Αυτόματη ονοματοδοσία με ημερομηνία/ώρα

## ❓ Συχνές Ερωτήσεις (FAQ)

**Ε: Γιατί μου λέει "Access Denied" κατά την αποθήκευση;**
Α: Επειδή το Windows είναι λίγο drama queen με τα δικαιώματα. Εκτελέστε το πρόγραμμα ως Administrator και θα συνέλθει.

**Ε: Μπορώ να αλλάξω τους βασικούς μισθούς;**
Α: Ναι! Απευθείας στο αρχείο `Wage_Config.cfg` ή μέσω του προγράμματος στις αναπτυσσόμενες λίστες.

**Ε: Τι κάνει το κουμπί "INFO";**
Α: Σας δείχνει πληροφορίες για το πρόγραμμα και τα στοιχεία επικοινωνίας του δημιουργού.

**Ε: Οι υπολογισμοί είναι 100% ακριβείς;**
Α: Βασίζονται στις επίσημες ασφαλιστικές και φορολογικές διατάξεις, αλλά συνιστάται επαλήθευση με λογιστή για κρίσιμες περιπτώσεις.

## 🔧 Αντιμετώπιση Προβλημάτων

### 🚫 Το πρόγραμμα δεν ανοίγει
- Ελέγξτε αν το antivirus μπλοκάρει το αρχείο
- Δοκιμάστε εκτέλεση ως Administrator
- Ελέγξτε για μηνύματα UAC (User Account Control)

### 🐌 Το πρόγραμμα "κολλάει"
- Δοκιμάστε Ctrl + Alt + Del → Διαχείριση Εργασιών → Τερματισμός Εργασίας
- Επανεκκινήστε το πρόγραμμα

### 🔄 Καθαρή Επανεγκατάσταση
1. Απεγκαταστήστε από τον Πίνακα Ελέγχου
2. Διαγράψτε τον φάκελο εγκατάστασης
3. Διαγράψτε το αρχείο `Wage_Config.cfg` (αν υπάρχει)
4. Εγκαταστήστε ξανά
